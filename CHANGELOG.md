### 变更

- 网页新增 Google Play 下载入口，并保留 APK 下载作为备用方式
- README 的项目入口改为文字超链接，同时补充 Google Play 下载入口
- 网页中的 Android APP 下载链接改为 GitHub latest Release 重定向地址，避免每次发版手动改版本号链接
- README 中的 APK 下载链接同步改为 latest Release 重定向地址，保持文档与网页一致
- 网页两个 Android 下载按钮新增悬浮二维码预览，鼠标移动到按钮时可在按钮上方查看二维码
- 二维码层不拦截按钮点击，并且仅在支持悬浮的设备展示，手机端点击按钮会直接下载 APK
